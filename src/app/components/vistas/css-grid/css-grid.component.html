Declaración Información
grid-template-columns: La propiedad grid-template-columns define el nombre de las líneas y las funciones de tamaño de línea de grid columns.
grid-template-rows: La propiedad grid-template-rows define el nombre de las líneas y las funciones de tamaño de línea de grid rows.
grid-auto-rows: La propiedad grid-auto-rows de CSS especifíca el tamaño de una nueva fila creada de forma implícita.
gap: Espacio entre los elementos tanto vertical como horizontal.
column-gap: Espacio entre las columnas.
row-gap: Espacio entre las filas.
justify-items: La propiedad CSS justify-items define cómo el navegador distribuye el espacio entre y alrededor de los items flex, a lo largo del eje principal de su contenedor.
align-items: La propiedad align-items indica la alineación de un elemento dentro del bloque que lo contiene.
align-self: // justify-self: La propiedad CSS que alinea los elementos flexibles de la línea flexible actual, reemplazando el valor de align-items.
grid-column: // grid-row: Un grid es una pista vertical en un CSS Grid Layout, y es el espacio entre dos líneas verticales de cuadrícula.
